- 环境准备
    * 安装配置Anaconda环境
    * pip install flask
- 运行
    * 在项目目录下，直接运行python app.py
    * 工作空间为files目录
    * 默认端口号为2000，可以更改
    * 浏览器访问127.0.0.1:2000

- 功能
   1. 输入文件路径：a.txt, 编辑器中显示出文件内容。
   2. 点击新建按钮，输入文件名，就可以在files/目录下新建一个对应的文件。
   3. 点击保存，保存当前编辑器中的内容到对应的文件。
   4. 最下面是一个webshell。
